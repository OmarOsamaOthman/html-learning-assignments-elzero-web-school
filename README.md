# HTML Learning Assignments

This repository contains my **HTML assignments and practice projects** completed during my learning journey with the **Elzero Web School HTML course**.

The goal of these assignments is to practice the fundamentals of HTML and understand how to build well-structured web pages using modern semantic elements.

---

## 📚 Topics Covered

Throughout these assignments I practiced many important HTML concepts including:

- HTML page structure
- Headings and paragraphs
- Lists (ordered, unordered, description)
- Links and navigation
- Images and figures
- Tables (rowspan, colspan, captions)
- Semantic HTML elements
- Forms and input types
- Audio and video embedding
- Iframes
- Accessibility basics

---

## 📂 Assignments Included

This repository includes multiple assignments such as:

- Personal information page
- Nested lists and structured content
- Complex tables with rowspan and colspan
- Technology blog layout using semantic HTML
- Forms with validation and different input types
- Media elements (audio & video)
- Search, file upload, and date inputs
- Embedded web pages using iframes

Each assignment focuses on practicing specific HTML concepts step by step.

---

## 🧠 What I Learned

During this practice I learned how to:

- Structure a complete HTML document
- Organize content using semantic HTML
- Create tables and forms
- Embed multimedia content
- Build clean and readable markup
